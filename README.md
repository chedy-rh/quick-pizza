# Quick Pizza

A small app to order some pizzas inspired by a Udemy course project and enhanced beyond the original implementation. Upgraded from Vite v4 to v7, Tailwind CSS v3 to v4 , and latest React Router, with code refactoring and UI improvements..

## How to use the app

1- install dependencies using : **pnpm install**  
2- run using : **pnpm dev**
